# Contribution

This app is developed as an open-source project, so anyone can contribute. We would appreciate your contribution in the following ways.

<LinkCard
  title="GitHub Repository"
  description="Check out the GitHub repository for TutoriaLLM."
  href="https://github.com/TutoriaLLM/TutoriaLLM"
/>

<Aside type="caution" title="Attention!">
  When submitting a pull request, please create an Issue first and submit a pull
  request to the Dev branch.
</Aside>

## Reporting Bugs

You can report bugs in TutoriaLLM via the Issues on the GitHub repository.

<LinkCard
  title="GitHub Issue"
  description="Check out the Issues section of the TutoriaLLM GitHub repository."
  href="https://github.com/TutoriaLLM/TutoriaLLM/issues"
/>

## Proposing Features

You can propose new features via the Issues on the GitHub repository.

<LinkCard
  title="GitHub Issue"
  description="Check out the Issues section of the TutoriaLLM GitHub repository."
  href="https://github.com/TutoriaLLM/TutoriaLLM/issues"
/>

## Contributing Code

We accept code contributions via Pull Requests on the GitHub repository.
After reviewing the repository, please submit a pull request to the Dev branch.

<LinkCard
  title="GitHub Repository"
  description="Check out the GitHub repository for TutoriaLLM."
  href="https://github.com/TutoriaLLM/TutoriaLLM"
/>

## Contributing to Localization

We would greatly appreciate contributions to the localization of TutoriaLLM.
Language settings can be found in `/apps/frontend/src/i18n/`.
After reviewing the repository, please submit a pull request to the Dev branch.

<LinkCard
  title="/apps/frontend/src/i18n/ Repository"
  description="Check out the GitHub repository for TutoriaLLM."
  href="https://github.com/TutoriaLLM/TutoriaLLM/tree/main/apps/frontend/src/i18n/"
/>

## Improving Documentation

You can improve the documentation by submitting a pull request to the documentation repository.
After reviewing the repository, please submit a pull request directly to the Main branch.

<LinkCard
  title="TutoriaLLM Documentation Repository"
  description="Check out the documentation repository for TutoriaLLM."
  href="https://github.com/TutoriaLLM/TutoriaLLM-web"
/>

## Financial Support

The costs of operating TutoriaLLM are personally funded. If you would like to provide financial support, please feel free to contact us.
