# 貢献

このアプリは、オープンソースで開発されているため、誰でも貢献することができます。以下の方法で貢献していただけると幸いです。

<LinkCard
  title="GitHub リポジトリ"
  description="TutoriaLLM の GitHub リポジトリを確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM"
/>

<Aside type="caution" title="注意！">
  プルリクエストを送信する際は、Issueを作成してから、Devブランチに対してプルリクエストを送信してください。
</Aside>

## バグの報告

TutoriaLLM 自体のバグの報告は、GitHub リポジトリの Issue にて受け付けています。

<LinkCard
  title="GitHub Issue"
  description="TutoriaLLM の GitHub リポジトリの Issue を確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM/issues"
/>

## 機能の提案

新しい機能の提案は、GitHub リポジトリの Issue にて受け付けています。

<LinkCard
  title="GitHub Issue"
  description="TutoriaLLM の GitHub リポジトリの Issue を確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM/issues"
/>

## コードの貢献

コードの貢献は、GitHub リポジトリの Pull Request にて受け付けています。
リポジトリを確認後、Devブランチに対してプルリクエストを送信してください。

<LinkCard
  title="GitHub リポジトリ"
  description="TutoriaLLM の GitHub リポジトリを確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM"
/>

## 多言語化への貢献

TutoriaLLM の多言語化に貢献していただけると幸いです。
言語設定は、`/apps/frontend/src/i18n/` にあります。
リポジトリを確認後、Devブランチに対してプルリクエストを送信してください。

<LinkCard
  title="/apps/frontend/src/i18n/ のリポジトリ"
  description="TutoriaLLM の GitHub リポジトリを確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM/tree/main/apps/frontend/src/i18n/"
/>

## ドキュメントの改善

ドキュメントのリポジトリから、プルリクエストを送信することで、ドキュメントの改善に貢献していただけます。
リポジトリを確認後、Mainブランチに対して直接プルリクエストを送信してください。

<LinkCard
  title="TutoriaLLMドキュメントのリポジトリ"
  description="TutoriaLLM のドキュメントリポジトリを確認してください。"
  href="https://github.com/TutoriaLLM/TutoriaLLM-web"
/>

## 金銭的な支援

TutoriaLLM の運営にかかる費用は個人で負担しています。もし、金銭的な支援をしていただける場合は、ご連絡いただけると幸いです。
